# Estrutura do projeto

```
├── POO --> Pasta onde ficará tudo relacionado a POO
│   └── Avaliação --> Pasta do projeto
│       ├── bin --> Simplesmente ignore essa pasta
│       ├── lib --> Pasta onde o JavaFX vai ficar armazenado
│       └── src --> Pasta onde ficará o código-fonte da aplicação Java
│           ├── control --> Irá armazenar o código relacionado com a parte de control
│           ├── model --> Irá armazenar o código relacionado com a parte de model
│           └── view --> Irá armazenar o código relacionado com a parte de view
└── PW --> Pasta onde ficará tudo relacionado a PW
```
