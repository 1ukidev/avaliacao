# Estrutura do projeto

```
├── POO --> Pasta onde ficará tudo relacionado a POO
│   └── Avaliação --> Pasta do projeto / Obs: essa pasta que você deve abrir no VSCode
│       ├── bin --> Simplesmente ignore essa pasta
│       ├── lib --> Pasta onde o JavaFX vai ficar armazenado
│       └── src --> Pasta onde ficará o código-fonte da aplicação Java
│           ├── control --> Pasta que irá armazenar o código relacionado com a parte de control
│           ├── model --> Pasta que irá armazenar o código relacionado com a parte de model
│           └── view --> Pasta que irá armazenar o código relacionado com a parte de view
└── PW --> Pasta onde ficará tudo relacionado a PW
```
